# Cub3D

## Description

Cub3D is a graphical project inspired by the legendary 1990s game Wolfenstein 3D, designed to explore the foundations of 3D rendering using a technique called raycasting.
The program takes a .cub file as an argument, which defines textures, colors, and a grid-based map layout.
Using the MiniLibX library, the project handles real-time window management and user inputs, such as movement and rotation.

## Instructions

### Compilation

Compile the project using:

```bash
make
```

### Execution

Run the program by passing no arguments:

```bash
./Cub3D <file.cub>
```
